🚀 Porichat – Anonymous Chat Platform

A modern anonymous chat platform built with Next.js, where users can connect and chat with strangers instantly.

---

✨ Features

- 🔐 Anonymous chat system
- ⚡ Real-time messaging
- 🎨 Modern UI (Tailwind CSS)
- 🌐 Full-stack (Frontend + Backend)
- 🗄️ Database integration (Prisma)

---

📁 Project Structure

Porichat/
│── src/            # Frontend (Next.js)
│── server/         # Backend logic
│── prisma/         # Database schema
│── public/         # Static files
│── db/             # Database related files
│── package.json    # Dependencies

---

🛠️ Tech Stack

- ⚛️ Next.js
- 🟨 JavaScript / TypeScript
- 🎨 Tailwind CSS
- 🗄️ Prisma ORM
- 🌐 Node.js

---

⚙️ Installation

git clone https://github.com/Asif-Aviator-predict/Porichat.git
cd Porichat
npm install

---

▶️ Run Locally

npm run dev

👉 Open: http://localhost:3000
